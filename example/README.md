# 目录结构
```
├── biz		业务逻辑
├── config	配置、全局函数、数据模型
├── bsvs		base services 基础服务
├── vue		前端vue代码
└── web		对外Web服务
```

# biz 业务逻辑

## 原则
- 校验输入数据
- 只与基础服务或数据库相关
- 返回结果或错误
- 变量、常量、错误等统一定义到bizVer.go文件
- 重要逻辑必须配有test方法

## 结构
```shell
# 验证输入数据

# 基础服务调用、数据库调用

# 返回数据或错误
```

# base services 基础服务

## 原则
- 每个基数服务完全独立开发
